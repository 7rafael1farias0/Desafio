🚀 Desafios de Lógica: Resolvendo Problemas Práticos
Este repositório contém resoluções para três desafios fundamentais de lógica de programação, focados em manipulação de variáveis, estruturas condicionais e laços de repetição.

🎓 1. Sistema de Status Acadêmico

O Desafio: Analisar a média de um aluno e decidir seu destino escolar. 🏫

Tabela de Resultados 📋:

🔴 Média < 5: Reprovado (Precisa de mais foco! 📚)

🟡 Média entre 5 e 6.9: Recuperação (Quase lá, não desista! 💪)

🟢 Média ≥ 7: Aprovado (Parabéns! Férias garantidas! 😎)

🔢 2. O Grande Duelo: Par vs. ÍmparO Desafio:

Receber uma lista de $N$ números e filtrar os "protagonistas" de cada categoria.
🌟Tabela de Resultados 📋:

👑 Maior Par: Identifica o maior valor entre os números pares (O topo! ⬆️)

💎 Menor Ímpar: Identifica o menor valor entre os números ímpares (A base! ⬇️)

🔄 Quantidade N: Processa quantos números você desejar informar.

💰 3. Calculadora de Salário Líquido

O Desafio: Calcular o valor final a ser transferido para um funcionário, considerando impostos e benefícios. 💸

Tabela de Impostos 📉:

🟣 Até R$ 1.100,00: Imposto de 5% (Menor alíquota)

🔵 De R$ 1.100,01 a R$ 2.500,00: Imposto de 10% (Alíquota média)

🟠 Maior que R$ 2.500,00: Imposto de 15% (Maior alíquota)
