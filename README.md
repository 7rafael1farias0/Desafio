# 🚀 Desafios de Lógica: Resolvendo Problemas Práticos
Este repositório contém resoluções para três desafios fundamentais de lógica de programação, focados em manipulação de variáveis, estruturas condicionais e laços de repetição.

---

## 💰 1. Calculadora de Salário Líquido
**O Desafio:** Calcular o valor final a ser transferido para um funcionário, considerando impostos e benefícios. 💸

**Tabela de Resultados 📋:**
* 🟣 **Até R$ 1.100,00:** Imposto de **5%** 
* 🔵 **De R$ 1.100,01 a R$ 2.500,00:** Imposto de **10%** 
* 🟠 **Maior que R$ 2.500,00:** Imposto de **15%**

---

## 📊 2. Analisador de Extremos: Pares e Ímpares
**O Desafio:** Receber uma lista de N números e filtrar os "protagonistas" de cada categoria. 🌟

**Tabela de Resultados 📋:**
* 👑 **Maior Par:** Identifica o maior valor entre os números pares.
* 💎 **Menor Ímpar:** Identifica o menor valor entre os números ímpares .
* 🔄 **Quantidade N:** Processa quantos números você desejar informar.

---

## 🎓 3. Sistema de Status Acadêmico
**O Desafio:** Analisar a média de um aluno e decidir seu destino escolar. 🏫

**Tabela de Resultados 📋:**
* 🔴 **Média < 5:** Reprovado
* 🟡 **Média entre 5 e 6.9:** Recuperação
* 🟢 **Média ≥ 7:** Aprovado

---

## 🛠️ Tecnologias e Ferramentas
* **Linguagem:** *JavaScript* 💻
* **Lógica:** Estruturas de repetição e condicionais. 🧠
